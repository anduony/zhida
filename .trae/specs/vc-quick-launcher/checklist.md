# Checklist

- [ ] 程序能正常编译为 Windows 可执行文件（.exe），无编译错误和警告
- [ ] 软件名称在所有界面位置统一显示为"直达"
- [ ] 程序启动后在系统托盘中显示"直达"图标
- [ ] 托盘图标右键菜单包含"显示窗口"、"设置"、"退出"选项
- [ ] 托盘图标左键双击可显示主窗口
- [ ] 程序最小化或窗口隐藏后仍通过系统托盘后台运行
- [ ] 默认快捷键 Alt+X 可呼出主窗口，再次按下可隐藏
- [ ] 主窗口无边框、四角圆角（12px）、带投影阴影
- [ ] 主窗口显示/隐藏有淡入淡出动画（约 150ms）
- [ ] 主窗口在屏幕中央显示，宽度约 640px
- [ ] 窗口失去焦点时自动隐藏
- [ ] 卡片以 4 列网格布局展示，每行 4 张卡片
- [ ] 卡片响应式布局：不足 4 张时自动调整列数并居中
- [ ] 卡片默认样式为白色背景（圆角矩形）、左侧 4px 彩色竖条
- [ ] 卡片标题为加粗字体
- [ ] 卡片包含图标和名称，点击可打开对应目标（文件夹/程序/文件）
- [ ] 卡片悬停时有平滑背景色过渡动画（约 200ms）
- [ ] 卡片点击时有微缩放反馈效果
- [ ] 窗口顶部有"直达"品牌标题区域
- [ ] 卡片区域有 16px 内边距，卡片间距 12px
- [ ] 至少内置 5 种配色方案（极简白、暗夜黑、石墨灰、深海蓝、暖沙色），切换即时生效
- [ ] 设置窗口中透明度滑块可调节主窗口透明度（30%-100%），实时预览
- [ ] 设置窗口中可录入自定义快捷键，保存后生效
- [ ] 设置窗口中可勾选/取消开机启动，重启电脑后生效
- [ ] 设置窗口中有配色方案选择器，选择后即时预览
- [ ] 设置按钮（托盘菜单 + 主窗口入口）可正常弹出设置窗口
- [ ] 所有配置保存到 %APPDATA%\直达\config.json，重启程序后保留
- [ ] 退出菜单项可完全退出程序并清理系统托盘图标
- [ ] 程序嵌入 AI 生成的多尺寸图标（16/32/48/256）
- [ ] 窗口在不同 DPI 缩放下显示正常
- [ ] 字体使用 Microsoft YaHei / Segoe UI 系统字体
